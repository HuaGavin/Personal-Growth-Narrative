# 🌱 个人成长史 · 寻找真实的自己

> 一个基于 AI 的个人成长史书写工具。通过系统回顾人生经历，发现行为模式、核心价值观与成长主线。

## ✨ 是什么

个人成长史书写是一种通过系统回顾人生经历来深度了解自我的方法。

本项目将这一过程与 AI 深度结合，融合叙事心理学、社会学、人类学等多学科视角，引导你完成从"回忆"到"理解"的完整旅程。


## 🚀 功能特性

- **🧭 四阶段结构化引导**
  - 📊 阶段划分：将人生划分为 6-8 个阶段，找到每个时期的关键词
  - 📖 事件回顾：逐阶段深度探索 3-5 件影响最深的事件
  - 🔍 跨阶段整合：发现重复模式、情绪主题与成长弧线
  - 🎯 最终总结：提炼核心价值观、优势天赋与恐惧陷阱

- **📄 智能文档解析**
  - 支持上传 `.txt`、`.pdf`、`.docx` 文档
  - AI 自动提取关键成长线索，辅助回忆与梳理

- **🎨 沉浸式交互体验**
  - 🌓 暗色 / 亮色双主题，一键切换
  - 📈 阶段性进度指示，清晰掌控旅程节奏
  - 💬 对话式交互，每次只问 1-2 个问题，节奏舒缓
  - 📥 导出对话记录为 Markdown，随时复盘

- **⚡ 纯前端实现**
  - 单 HTML 文件，无需安装，浏览器直接打开
  - 仅需配置 DeepSeek API Key 即可使用

## 🏁 快速开始

1. 📦 克隆或下载本仓库
2. 🌐 用浏览器打开 `个人成长史_原型 0.0.4.html`
3. 🔑 在顶部输入你的 **DeepSeek API Key**
4. 🚀 点击"保存并开始"，开启你的成长史书写之旅

> 💡 需要 DeepSeek API Key？前往 [DeepSeek 开放平台](https://platform.deepseek.com/) 免费注册获取。

## 🛠️ 技术栈

- **🎨 前端**：原生 HTML5 + CSS3 + JavaScript（零框架依赖）
- **🤖 AI 引擎**：DeepSeek API（`deepseek-reasoner` 模型）
- **📄 文档解析**：PDF.js + Mammoth.js
- **📝 Markdown 渲染**：Marked.js

## 💝 对话原则

本项目遵循以下设计原则，确保你在安全、受尊重的环境中探索自我：

- **🤝 用户中心**：你是个人历史的最终权威，AI 只是辅助者与催化剂
- **🕊️ 非评判性**：对所有分享保持中立，营造安全的表达环境
- **💡 启发而非指导**：提供视角与问题，让你自己产生洞察
- **💗 重视情绪真实性**：而非记忆的精确性
- **🌱 聚焦成长**：围绕个人理解与未来，不沉溺于过去

## 🗺️ 成长史书写流程概览

```
┌─────────────────────────────────────────────────────┐
│  第一步：建立信任与划分阶段                            │
│  → 划分 6-8 个人生阶段（必须包含社会化前阶段）        │
│  → 以表格形式呈现，加强记忆                            │
├─────────────────────────────────────────────────────┤
│  第二步：逐阶段深度回顾                                │
│  → 选出 3-5 件影响最深的事件                          │
│  → 对每件事件探索：体验 → 驱动力 → 影响 → 成长视角    │
│  → 产出阶段分析报告（5 个核心问题）                    │
├─────────────────────────────────────────────────────┤
│  第三步：跨阶段整合                                    │
│  → 反复出现的关键词与情绪主题                          │
│  → 人际/工作/应对压力的生命模式                        │
│  → 社会化之前的天性特质回溯                            │
│  → 一句话总结成长弧线                                  │
├─────────────────────────────────────────────────────┤
│  第四步：最终总结                                      │
│  → 3 个核心价值观 · 3 个优势天赋 · 3 个恐惧陷阱       │
│  → 1 句话描述未来想成为的人                            │
└─────────────────────────────────────────────────────┘
```
